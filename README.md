# Sports Betting

## Description

Our app is designed to provide live win probabily modeling and player performance projections for daily NBA Matchups. The system ingests live NBA game data and runs predictive models to boradcast live odds updates to all user in real time. Anyone from sports bettors to casula fans are able to perform pre-game analysis and monitor probabilites throughout each game. This design is meant to provide a clear and interactive view of probabilites and statistics in one clean dashboard.

## Audience

Our ideal user are casual NBA fans and sports bettors who want a real-time view of matchup odds and player performance projections.

---

### Users

#### 1. NBA Fans (Viewers)
Are users whos main goal is to simply view live probabilties and player performance projections. 

**Can:**
- View live NBA games and matchups
- See player performance projections
- Track real-time win probabilites

#### 2. Sports Bettors
Are user whos main goal is to derive insight from live probabilites and player performance projections to guide betting decisions.

**Can:**
- Analyze player peformance projects and matchup probabilites
- Identify potential betting opportunities

---

This platform helps user to accomplish their goals by ingests live NBA game data, running predictive models to boradcast live odds updates to all user in real time all in on dashboard. This allows users to stay up to date and informed throughout the game for all their needs.
